# To-Do App


This project was using boilerplate from [superplate](https://github.com/pankod/superplate) .

## Getting Started

To-Do App is an app to create your own to-do-list. This app created using React, Redux, and Tailwind CSS.

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```
